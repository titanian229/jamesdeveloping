Hi there! I'm a full stack developer, based in Toronto. I'm focused on back end software development but growing my front end experience.

I'm proficient in Python, particularly using Numpy and SciPy. I'm skilled in Javascript, CSS, and HTML for web development. I'm also experienced with using SQL and it's variants for database management. I've built applications from a database management android application used in managing ecommerce orders to an interface for bulk creation of events using the Google Calendar API.

I first encountered programming by exploring it myself, creating software to automate my life and work and improve things I encountered day to day. Mapping my routes for kayak trips, automating the bulky process of making gCal events, creating android applications to take my programs with me on the go. I taught myself by exploring MAN pages, reading online documentation, picking apart things and playing with them. I found my passion in it. I'm always growing my skillset through courses and online programs, most recently a University of Toronto coding bootcamp.

Contact me, and let's see how we can best put my passion to work for you!
